# TicTacToe
The application allows you to play games between a Pastafarian and a Ghost. Who will win the game? Game is based on tic tac toe.
Live demo [_here_](https://eparzych.github.io/TicTacToe)

## Technologies Used
- React
- SASS
- react-icons
- creat-react-app
- gh-pages
